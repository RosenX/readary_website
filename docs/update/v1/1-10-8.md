# 1.10.8

## 功能
支持自动归档，可以
1. 按发布时间归档
2. 按某个关键词组合归档
3. 按某个主题归档
4. 以上方法的组合
以上自动归档均可以指定RSS源(不限/指定集合/排除部分)

## 体验
1. 优化主题显示，不限条件不显示
2. 优化更新机制，没有解析完成不显示在列表中
3. 应用新增关键词时，如果满足主题条件，自动打上主题标签。应用新增关键词/主题时，如果满足自动归档条件，自动归档。
4. 优化聚焦页面卡片上页面图片显示
5. 优化文章上标签显示。主题与关键词同时出现时，如果关键词包含在主题中则不显示，避免太多标签。
6. 优化正文阅读标题显示，最多保留3行
7. 优化关注页面关键词显示
   
## Bug修复
1. 修复修改关键词或者分组时，主题条件不变化问题
